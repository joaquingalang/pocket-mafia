## About the Project  

Pocket Mafia is an assistive tool designed to serve as the moderator for the classic Mafia party game.
Play face-to-face with your friends, configure game details, select roles, and have fun!

## Features

The Pocket Mafia mobile application includes four main features:  

- **Role Selection** – Choose from over 12 roles to add to the game. Role's will be randomly assigned when the game starts.
- **Moderated Gameplay** – Pocket Mafia automatically moderates gameplay. Keeping track of discussion time, tallying votes, and prompting actions from special roles during the night. No moderator needed. Everyone gets to play.

## Tech Stack  

- **Flutter** – Cross-platform frontend framework (Android-focused)
